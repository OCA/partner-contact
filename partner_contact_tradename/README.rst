=================
Partner tradename
=================

This module adds the field tradename for partner and partner views.

Installation
============

To install this module, you need to:

* Add the module to your addons path.
* Install the module as usual.

Configuration
=============

This module doesn't requiere any configuration.

Usage
=====

After the partner name you will see a new field to save the partner tradename.

For further information, please visit:

* https://www.odoo.com/forum/help-1
* https://github.com/OCA/partner-contact/
